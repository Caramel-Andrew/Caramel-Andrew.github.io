システム サービス ディレクター
Andrew Hewitt
 アンドリュー・ヒューイット
英国にてPascal / Visual Basic / C# / C++などの講師を3年間勤めたのち 自動販売機開発会社へ入社。エンジニア / グラフィックデザイナー / ビデオエディターとして従事。 自動販売機からの売上・各種情報をデータベース化するシステムや、 ロンドン地下鉄に設置された、「Cadbury Chocolate（キャドバリー）」のタッチスクリーン式自動販売機の開発を行う。
また、弁護士向けCRM（顧客管理システム）開発のプロジェクトリーダーを務めたのち、2010年来日。 日本ではこれまでに、Android向けゲーム、スマートフォンアプリ、CRM、スマートスピーカーや AI通話システムを手がける。

《開発言語/環境》
言語：C++、C#、VB、Java、JavaScript、Python、Ruby、Skala、Kotlin、Pascal、PHP、HTML
データベース：MongoDB、MS SQL、MySQL
統合開発環境：Visual Studio、Visual Studio Code、Unity、Unreal Engine、AppGameKit、GameGuru
デバイス：Windows、Linux、Android、iOS、Web、スマートスピーカー、Oculus、ゲームコンソール、Raspberry Pi、Arduino、Beagle Bone

《実績》
スマートフォンゲーム / アプリ
CRM（顧客管理システム）
スマートスピーカーを用いた物流トラッキングシステム
AI電話（通話式予約・注文システム）
タッチスクリーン式「Cadbury Chocolate（キャドバリー）」自動販売機
自動運転ラジコンカー　など
