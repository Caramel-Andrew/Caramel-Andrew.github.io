After Effects Plugins

miracle Color Selector

for After Effects

miracleColorSelector is a plug-in for extracting selected colors; An essential task for animation production.
Both MacOS and Windows After Effects are supported,
allowing you to work seamlessly across operating systems.
Free for both personal and commercial use.

DOWNLOAD (Windows)
DOWNLOAD (Mac)

↓ How To Use

miracleColorSelector

After launching After Effects, select the layer to which you want to apply miracleColorSelector and click Effects/CARAMEL.codeVFX/miracleColorSelector

When the plug-in is applied, the selected colors are extracted.
Up to 10 colors can be selected.
Using the sliders, the range of color can be widened and softness applied.

Enable Effect

If checked, only the colors set in Colors 1 - 10 will be extracted. If the Enable color item below * is not checked, the process will not be reflected.

Enable color

Check the box to include this color in the extraction.

Color

Specifies the color to be extracted. Colors can be specified directly from the layer using an eyedropper.

Range

Specifies the range of color matching. A small value reduces the acceptable matches, while a larger value increases it.

Softness

Adjust the degree of blurring around the boundaries of the selected colors.

miracle Smoother

for After Effects

miracleSmoother is a one-click plugin to smooth out jagged edges in your images or animations.
Both MacOS and Windows After Effects are supported,
allowing you to work seamlessly across operating systems.
Free for both personal and commercial use.

DOWNLOAD (Windows)
DOWNLOAD (Mac)

↓ How To Use

miracleSmoother

After launching After Effects, select the layer on which you want to apply miracleSmoother and click Effects/CARAMEL.codeVFX/miracleSmoother

Use Color Key

If checked, the keying process will be carried out in the selected color.

Alpha Key

Select the colors for the keying process here.

Smoother Range

Adjusts the threshold for neighbouring color differences when applying anti-aliasing. The lower the value here, the more anti-aliased the image will be, even in areas with small color differences.
Normally, the default value of 1.0 can be used.

Line Weight

Adjusts the width of the line to be anti-aliased.

INSTALLATION

After downloading from the button above, unzip the file.

Extract the plugin file to the C:\Program Files\Adobe\Adobe After Effects 2023\Support Files\Plug-ins\Effects directory, or the C:\Program Files\Adobe\Common\Plug-ins\7.0\MediaCore directory.

※Each should be placed in the folder of the After Effects version you are currently using.
(After Effects 2023 in the picture above).

Verified versions

After Effects 2023 (23.x)
Windows 10 / 11 (64-bit) / macOS 10.15 - 13

After Effects 2022 (22.x)
Windows 10 / 11 (64-bit) / macOS 10.15 - 12
