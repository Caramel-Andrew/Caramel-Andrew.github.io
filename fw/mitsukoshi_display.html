日本橋三越本店 本館1Fインタラクティブディスプレイ

Mitsukoshi de Asobo Interactive Display from YOYOmiracle, on Vimeo.

日本橋三越本館1Fウィンドウ
インタラクティブディスプレイ
2016.4.19 - 5.7

System Planning / Program / Motion Graphics / Sound Design

2016年4月19日から5月7日まで東京・日本橋三越本店 本館1階路面ディスプレイにておよそ3週間の間 展示されたインタラクティブディスプレイ。全4面の中央通り沿いの路面ディスプレイのうち真ん中の2つにインタラクティブ要素を取り入れ展示しました。48インチモニタを縦置きにし3面つなげることにより、より大きなディスプレイ展開ができ、向かって右側に木琴やハープを奏でることのできるシステム、左側にはドラムや民族打楽器を叩いたような感覚になれる展示となっています。
